Optimizador IO

Interfaz web para resolver problemas de transporte con métodos interactivos.

Escoge el tamaño del problema, completa costos, oferta y demanda, y resuelve con Esquina Noroeste, Costo Mínimo o Vogel.

Proyecto de escritorio original: GitHub.

Esta interfaz web está creada para ejecutar los métodos del problema en el navegador.
